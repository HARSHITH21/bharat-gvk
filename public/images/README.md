# Images Directory

This directory contains all the images used in the website.

## Image Structure

- `slider3.jpg`, `slider7.jpg`, `slider5.jpg` - Hero carousel images
- `img1.png` - About section image
- `service-1.jpg`, `service-2.jpg`, `service-3.jpg`, `service-4.jpg` - Service images
- `img2.jpg`, `img3.jpg`, `img4.jpg`, `img5.jpg`, `img6.jpg`, `img7.jpg`, `img8.jpg`, `img9.jpg` - Product images
- `imag1.jpg` through `imag15.jpg` - Application images
- `image6.png`, `image8.jpg`, `image9.jpg`, `image1.png`, `image2.jpg`, `image3.jpg`, `image4.jpg`, `image5.jpg` - Client logos

## Optimization Notes

- All images should be optimized for web use
- Use appropriate image formats (WebP for modern browsers, JPEG for photos)
- Implement lazy loading for better performance
- Use responsive images with srcset
